# Aptis Reading Practice Web Application

Ứng dụng web để ôn tập phần Reading của kỳ thi Aptis.

## Tính năng

- **Part 1**: Sentence Completion - Chọn từ điền vào chỗ trống
- **Part 2 & 3**: Text Cohesion - Sắp xếp câu theo thứ tự đúng
- **Part 4**: Opinion Matching - Khớp ý kiến với câu hỏi
- **Part 5**: Long Text Comprehension - Chọn heading cho đoạn văn

## Cách sử dụng

1. Chọn Part muốn làm
2. Chọn Đề (hoặc "Tất cả các đề")
3. Làm bài và nhấn "Check đáp án" để kiểm tra
4. Nhấn "Submit" để xem điểm chi tiết

## Công nghệ

- HTML5
- CSS3
- JavaScript (Vanilla)
