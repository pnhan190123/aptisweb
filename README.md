# Aptis Reading Practice Web App

Ứng dụng web để ôn tập kỳ thi Reading Aptis với 5 phần thi khác nhau.

## Cấu trúc đề thi

1. **Part 1: Sentence Completion** - Chọn từ phù hợp để điền vào chỗ trống (5 câu)
2. **Part 2: Text Cohesion** - Sắp xếp câu theo thứ tự đúng (1 đoạn văn)
3. **Part 3: Opinion Matching** - Nối quan điểm với người phù hợp (7 câu)
4. **Part 4: Long Text Comprehension** - Chọn tiêu đề cho từng đoạn văn (7 đoạn)
5. **Part 5: Short Text Comprehension** - Chọn các câu đúng (5 câu)

## Cách sử dụng

1. Mở file `index.html` trong trình duyệt
2. Làm bài từng phần theo thứ tự
3. Sử dụng nút "Previous" và "Next" để điều hướng
4. Nhấn "Submit" sau khi hoàn thành tất cả các phần
5. Xem kết quả và nhấn "Restart" để làm lại

## Tính năng

- Giao diện đẹp, hiện đại và responsive
- Lưu tiến độ làm bài khi chuyển phần
- Chấm điểm tự động sau khi submit
- Hiển thị kết quả chi tiết cho từng câu hỏi
- Drag & drop cho phần sắp xếp câu

## File cấu trúc

- `index.html` - File HTML chính
- `styles.css` - File CSS cho giao diện
- `script.js` - File JavaScript xử lý logic
- `data.js` - File chứa dữ liệu câu hỏi

## Chỉnh sửa dữ liệu

Bạn có thể chỉnh sửa file `data.js` để thêm hoặc thay đổi câu hỏi theo nhu cầu.
